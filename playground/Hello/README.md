# Hello
The likeable greetings program 

## Using
To use this program:

```bash

# MAIN
# ----

# You will need to put your name in the [--name] field

python .\hello.py --name=[NAME]




# EXAMPLE
# -------

python .\hello.py --name=Marcello

```
